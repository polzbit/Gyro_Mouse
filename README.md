**`Gyro_Mouse`**  is an Arduino based system I've made as a first project. this system controls windows cursor according to the system momentary velocity.

## Getting Started
```
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
```
First power on the arduino system then run **`mouse_server.py`** after bluetooth scan it will prompt the user to choose bluetooth COM port from a list. input the right COM and the server will start receiving data from the arduino.
